# Lho Music 🎵

Spotify-like music player PWA. Search & stream free music from Jamendo API.

## Setup

1. Get a free Jamendo Client ID: https://developer.jamendo.com/v3.0
2. Open `index.html`, find line: `const CLIENT_ID = 'YOUR_JAMENDO_CLIENT_ID';`
3. Replace with your Client ID, save
4. Open the GitHub Pages URL, add to home screen (PWA)

## Features

- Search songs/artists/genres
- Trending weekly
- Playlist (add/remove)
- Play/pause/next/prev/seek
- Now playing display
- 1 file, ~12KB, no backend
